## Running for check the functionality of the application through Docker. Not for Development

```bash
# Start Api and Database
$ docker-compose up

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migrations

```bash
# migration generation
$ npm run migration:generate --name=
```
